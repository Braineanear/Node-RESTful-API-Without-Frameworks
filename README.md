# NodeJS REST API Without Frameworks

> CRUD REST API using NodeJS without frameworks

```
# Routes
GET      /api/products
POST     /api/products
GET      /api/products/:id
PATCH      /api/products/:id
DELETE   /api/products/:id
```

## Usage

```
# Install dependencies
npm install

# Run in develpment
npm run dev

# Run in production
npm start
```
